# Product Overview

DemoLottie is an Android application that demonstrates interactive Lottie animations with a zipper overlay effect. The app creates a system-wide overlay that users can interact with by dragging a zipper animation to reveal content underneath.

## Key Features

- System overlay with SYSTEM_ALERT_WINDOW permission
- Interactive zipper animation that responds to touch gestures
- Foreground service for persistent overlay functionality
- Three-layer animation system (wallpaper, row content, zipper)
- Auto-lock and auto-complete animations based on drag threshold
- Full-screen overlay with edge-to-edge support
